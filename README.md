💳 Credit Card Approval Prediction

Credit Card Approval Prediction is a Machine Learning-based web application that predicts whether a credit card application will be approved or rejected based on applicant information. The project uses historical data to build a predictive model and provides an easy-to-use web interface for users.

 📖 Project Overview

The main objective of this project is to automate the credit card approval process using Machine Learning. The application analyzes applicant details and predicts the approval status, helping reduce manual effort and improve decision-making.

✨Features

- Predicts credit card approval status.
- User-friendly web interface.
- Machine Learning-based prediction.
- Fast and reliable results.
- Data preprocessing and model evaluation.

Installation

1. Clone the repository

```bash
git clone https://github.com/shaiksahistafathima/CREDIT-CARD-APPROVAL-PREDICTION.git
```

2. Move to the project directory

```bash
cd CREDIT-CARD-APPROVAL-PREDICTION
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

🚀How It Works

1. The user enters applicant details.
2. The input data is preprocessed.
3. The trained Machine Learning model predicts the approval status.
4. The result is displayed on the web page.
