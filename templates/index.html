Why did Tree-based Models score 100%?

Because the credit-card approval target labels were synthesized using a deterministic rule-based heuristic (e.g., checking if annual income is ≥ $120,000 AND years employed ≥ 2, etc.), tree-based classifiers (Decision Trees & Random Forests) were able to split the features exactly at those thresholds and reconstruct the mathematical logic perfectly on validation test sets.

Why did we select Random Forest? While a single Decision Tree can model these rules, it has high variance and is extremely prone to overfitting if new, noisy data is introduced. A Random Forest Classifier averages the decisions of 100 trees, reducing variance and ensuring a highly stable and robust model for credit operations.
